# Files and Folders Navigation

Labs covering essential Linux filesystem navigation and file management skills.

## Labs

| Lab | Description |
|-----|-------------|
| `linux_lab1_navigation_basics` | Moving around the filesystem with `cd`, `ls`, `pwd` |
| `linux_lab2_file_management` | Creating, copying, moving, and deleting files and directories |
| `linux_lab3_reading_files` | Reading file content with `cat`, `less`, `head`, `tail` |
| `linux_lab4_search_basics` | Searching for files and content with `find` and `grep` |
| `linux_lab5_scavenger_hunt` | Combined challenge: navigate a directory tree to find hidden clues |

Each lab folder contains a `README.md` with full instructions.
