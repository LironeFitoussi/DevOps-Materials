# Project Gamma

## Description
Data processing and analytics service

## Technology Stack
- Node.js 18
- Express
- MongoDB
- Bull (Job Queue)

## Setup Instructions
1. Install dependencies: npm install
2. Configure environment: cp .env.example .env
3. Start MongoDB: docker-compose up -d mongodb
4. Run service: npm start

## Features
- Real-time data processing
- Scheduled analytics jobs
- RESTful API
- WebSocket support
