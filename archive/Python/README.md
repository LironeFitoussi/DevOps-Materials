# Python

Python programming fundamentals, structured as progressive lab exercises.

## Topics

| # | Folder | Topic |
|---|--------|-------|
| 01 | `01_print_basics` | Printing output to the screen |
| 02 | `02_literals` | String, integer, float, and boolean literals |
| 03 | `03_operators` | Arithmetic and assignment operators |
| 04 | `04_variables` | Declaring and using variables |
| 05 | `05_comments` | Writing comments in code |
| 06 | `06_shortcuts` | Shorthand operators and expressions |
| 07 | `07_type_casting` | Converting between data types |
| 08 | `08_input` | Reading user input |
| 09 | `09_string_methods` | Common string methods and formatting |
| 10 | `10_comparison_operators` | Comparison and equality operators |
| 11 | `11_conditions` | `if`, `elif`, `else` statements |
| 12 | `12_conditions_intermediate` | Nested and complex conditions |
| 13 | `13_while_loops` | `while` loop basics |
| 14 | `14_for_loops` | `for` loop basics |
| 15 | `15_break_continue` | `break` and `continue` in loops |
| 16 | `16_logical_operators` | `and`, `or`, `not` |
| 17 | `17_combined_real_world` | Real-world exercises combining multiple concepts |
| 18 | `18_lists` | List creation and indexing |
| 19 | `19_lists_methods` | List methods (`append`, `remove`, `sort`, etc.) |
| 20 | `20_lists_variables_copies` | Mutable references and copying lists |
| 21 | `21_in_notin_combined` | `in` / `not in` operators |
| 22 | `22_lists_2d_3d` | Multi-dimensional lists |
| 23 | `23_lists_intermediate` | Intermediate list patterns |
| 24 | `24_functions_basics` | Defining and calling functions |
| 25 | `25_parameters` | Function parameters and arguments |
| 26 | `26_return` | Return values from functions |
| 27 | `27_list_as_argument` | Passing lists to functions |
| 28 | `28_scopes` | Variable scope (local vs global) |

## Lab Structure

Each topic folder contains:
- `TASKS.md` — exercises to complete
- `HINTS.md` — hints if you get stuck
- `src/` — starter code (where applicable)
- A `.pdf` version of the tasks
